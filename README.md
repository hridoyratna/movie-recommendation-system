# 🎬 Movie Recommendation System

A **content-based movie recommender** built with Python and Streamlit. Select any movie and instantly get 5 similar movie recommendations — complete with posters fetched live from the TMDB API.

## 🚀 Features

- Content-based filtering using movie metadata (genres, keywords, cast, crew, overview)
- Cosine similarity to find the top 5 most similar movies
- Live movie poster fetching via the TMDB API
- Simple and clean Streamlit UI with a 5-column poster layout
- Deployed-ready with Heroku configuration

---

## 🧠 How It Works

```
Raw TMDB Data (CSV)
        │
        ▼
  Data Preprocessing
  ├── Merge movies + credits datasets
  ├── Extract genres, keywords, top 3 cast, director
  ├── Combine into a single "tags" column
  ├── Apply stemming (PorterStemmer)
  └── Vectorize with CountVectorizer (top 5000 features)
        │
        ▼
  Cosine Similarity Matrix
        │
        ▼
  Streamlit App (app.py)
  ├── User selects a movie
  ├── Look up similarity scores
  ├── Return top 5 similar movies
  └── Fetch posters from TMDB API
```

## 🛠️ Setup & Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/movie-recommendation-system.git
cd movie-recommendation-system
```

### 2. Create and activate a virtual environment
```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# macOS/Linux
source .venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Generate the model files (if not included)
Open and run all cells in `Movie_Recommendation_System.ipynb`.  
This will generate `movies.pkl` and `similarity.pkl`.

### 5. Run the app
```bash
streamlit run app.py
```

Open your browser at `http://localhost:8501`

## 📓 Notebook Overview

`Movie_Recommendation_System.ipynb` covers the full ML pipeline:

1. Load and merge the two TMDB CSV datasets
2. Extract relevant columns: `movie_id`, `title`, `overview`, `genres`, `keywords`, `cast`, `crew`
3. Parse JSON-like string columns with Python's `ast` module
4. Keep only the top 3 cast members and the director
5. Remove spaces within names to treat them as single tokens (e.g., `Sam Worthington` → `SamWorthington`)
6. Create a combined `tags` column from all features
7. Apply **Porter Stemming** to normalize words
8. Vectorize using **CountVectorizer** (5000 features, English stop words removed)
9. Compute the **cosine similarity** matrix
10. Serialize results to `movies.pkl` and `similarity.pkl`
